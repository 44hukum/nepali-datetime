Command Line
============

Calendar
-------------
..  code-block:: bash

    $ python -m nepali_date calendar

Date
---------
..  code-block:: bash

    $ python -m nepali_date date


Current month
------------------
.. code-block:: bash

    $ python -m nepali_date month


Today's day
------------------
.. code-block:: bash

    $ python -m nepali_date day
